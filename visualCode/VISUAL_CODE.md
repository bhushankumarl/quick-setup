#### Keyboard : Auto Indent Shortcut
```
1) Preferences -> Settings -> Keyboard Shortcut
2) Search for :
`Format Document`
3) Double Click On It
4) Press : CTRL + SHIFT + Q
```

#### Keyboard : Copy Selected Line
```
1) Preferences -> Settings -> Keyboard Shortcut
2) Search for :
`Copy Line Down`
3) Double Click On It
4) Press : CTRL + D
```

#### Keyboard : Peek References
```
1) Preferences -> Settings -> Keyboard Shortcut
2) Search for :
`Peek References`
3) Double Click On It
4) Press : CTRL + SHIFT + BACKSPACE
```
